## Getting Started

### Prerequisites

To run this application, you need to have node.js and npm installed on your computer. To download the latest version of node.js, please visit: https://nodejs.org/en/download/

### Installation

To install the npm dependencies:

> npm i

To run the app:

> npm run dev
